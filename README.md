## Project Description
This is a Product Store web application built using React and Next.js.  
It displays products in card format and allows users to filter products by category.

## Technologies Used
- React
- Next.js
- Tailwind CSS
- Shadcn UI

## Features
- Product listing
- Category filter
- Clean UI using reusable components
